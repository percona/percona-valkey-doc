# What is Percona Packages for Valkey?

Percona Packages for Valkey are officially maintained and tested installation packages of Valkey that are built and distributed for specific operating systems.

The packages are designed to install Valkey easily on major Linux distributions and present a reliable, production-grade install method without relying on upstream community builds or Docker

Included in these packages are:

- Valkey server binaries
- CLI tools (like valkey-cli)
