# What Percona offers

Percona currently provides the following for Valkey and Redis users:

- 24/7 enterprise-grade expert support and direct contributions to Valkey, a Linux Foundation-governed, BSD-licensed, community-led project.
- 24/7 enterprise-grade support for existing Redis deployments, offering organizations strategic flexibility.
- Seamless migration to Valkey or optimization of your existing Redis infrastructure with unbiased expertise, mitigating risk and ensuring stability.

Explore the [Valkey Documentation by Topic](https://valkey.io/topics/) for in-depth guides on persistence, security, Lua scripting, modules, and more. Key references include the [Commands Reference](https://valkey.io/commands) for details on every command, and the [FAQ](https://valkey.io/topics/faq) for answers to common questions.
